# anchor/

This folder stores visual anchor resources for VEGA.

## Recommended setup

- Download `35_anchor.pt` (precomputed anchor features) from the Google Drive link in the root [`README.md`](../README.md).
- Place it at:

```text
anchor/35_anchor.pt
```

## Alternative setup

1. Download `35_anchor.zip` from the link in the root [`README.md`](../README.md).
2. Extract it to:

```text
anchor/35_anchor/
```

3. Build/save anchor cache to `anchor/35_anchor.pt`.
